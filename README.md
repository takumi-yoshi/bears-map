# 札幌市ヒグマ出没マップ
札幌市が公開している2016年から2025年までのヒグマ出没情報を地図上で可視化したWEBGISです。

既にGoogleマップ上で見えるものがありますが、過去のデータは閲覧できなかったり、フィルター機能は無かったので作成してみました。

また参考に、2004年（約20年前）の第6回基礎調査 動物分布調査 哺乳類分布調査において生息が確認されたクマ類の全国分布メッシュデータを追加してます。

## 概要
・ヒグマ出没地点をポイントで表示

・2004年基礎調査のクマ出没メッシュをポリゴンで表示

・年別・月別・時間帯別のフィルタリング機能

・出没件数の表示

・背景地図の切り替え

## 注意事項
本サイトは公開データを基に作成しています。

データの最新性・正確性・完全性を保証するものではありません。

実際のヒグマ出没状況の確認は、札幌市公式サイトをご参照ください。
